# TK-11 — Активность кнопки "Сохранить" при вводе недопустимого количества символов в поле "Имя"

**Предусловие:** Открыто окно редактирования профиля.

## Шаги
1. Кликнуть на строку ввода "Имя".
2. Ввести в поле 1 символ — “И”.
3. Кликнуть на строку ввода "Занятие".
4. Ввести “Художник”.

## Ожидаемый результат
Кнопка “Сохранить” не активна.
