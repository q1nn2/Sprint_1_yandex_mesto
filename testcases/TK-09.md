# TK-09 — При пустом поле "Имя" кнопка "Сохранить" не активна

**Предусловие:** Открыто окно редактирования профиля.

## Шаги
1. Очистить поле “Имя”.

## Ожидаемый результат
Кнопка "Сохранить" не активна.

## Статус
🔵 Blocked

## Причина блокировки
Невозможно выполнить шаг, т.к. окно редактирования открывается с пустыми полями, а по требованиям при открытии значения должны быть предзаполнены.
