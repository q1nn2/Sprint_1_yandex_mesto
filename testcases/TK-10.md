# TK-10 — Активность кнопки "Сохранить" при вводе недопустимого количества символов в поле "Занятие"

**Предусловие:** Открыто окно редактирования профиля.

## Шаги
1. Кликнуть на строку ввода "Имя".
2. Ввести “Иван”.
3. Кликнуть на строку ввода "Занятие".
4. Ввести “Х” (1 символ).

## Ожидаемый результат
Кнопка “Сохранить” не активна.
