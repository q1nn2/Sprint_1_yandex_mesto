# B13 — Не скроллятся галерея с карточками мест

**Приоритет:** Критический  
**Статус после ретеста:** ✅ Закрыт

## Предусловия
1. Открыта главная страница.
2. Количество добавленных мест больше трёх.
3. Если все карточки вмещаются в видимую часть страницы — изменить масштаб, чтобы они не вмещались.

## Шаги воспроизведения
1. Прокрутить колесико мышки для скролла галереи с карточками мест.

## Ожидаемый результат
Контент скроллится, отображаются остальные места.

## Фактический результат (до исправления)
Контент НЕ скроллится.
