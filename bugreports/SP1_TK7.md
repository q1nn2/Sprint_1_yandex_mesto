# SP1_TK7 — Не отображается сообщение об ошибке при вводе более 200 символов в поле «Занятие»

**Приоритет:** Низкий  
**Статус:** Открыт

## Предусловия
1. Перейти на тестовый стенд: https://code.s3.yandex.net/qa/files/mesto/index.html  
2. Открыть окно редактирования профиля (карандаш)

## Шаги воспроизведения
1. Кликнуть в поле «Занятие»
2. Вставить строку длиной 201 символ

## Ожидаемый результат
Под полем отображается красный текст об ошибке “Недопустимое количество символов”

## Фактический результат
При вводе 201 символа система автоматически обрезает значение до 200 символов. Сообщение об ошибке не отображается.
