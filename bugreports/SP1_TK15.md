# SP1_TK15 — Не отображается сообщение об ошибке при вводе более 40 символов в поле «Имя»

**Приоритет:** Низкий  
**Статус:** Открыт

## Предусловия
1. Перейти на тестовый стенд: https://code.s3.yandex.net/qa/files/mesto/index.html  
2. Открыть окно редактирования профиля (карандаш)

## Шаги воспроизведения
1. Кликнуть в поле «Имя»
2. Вставить строку длиной 41 символ

## Ожидаемый результат
Под полем отображается красный текст об ошибке “Недопустимое количество символов”

## Фактический результат
При вводе 41 символа система автоматически обрезает значение до 40 символов. Сообщение об ошибке не отображается.
